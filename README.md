# Wave Audio File Generator

## Description
This C++ program generates various types of sound waves and writes them to a WAV file. It supports wave types like square, sawtooth, triangle, sine, white noise, brown noise, and chirp. 

Additional features include low-pass filtering, amplitude modulation, and amplitude envelope application.

## Installation
1. Clone the repository: `git clone [repository URL]`
2. Ensure you have a C++ compiler installed.
3. Compile the program: `g++ -o wavegen WriteWaveFile.cpp`
